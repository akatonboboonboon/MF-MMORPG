Material Frontier Online - Phase 1 Technical Prototype
======================================================

起動方法
--------
1. MFO-Phase1.exe を実行します。
2. Windows SmartScreen が表示された場合は、ファイルの配置元とハッシュを確認してから実行してください。
3. 終了はウィンドウ右上の閉じるボタン、または Alt+F4 です。

操作
----
- 移動: WASD / ゲームパッド左スティック
- 照準: マウス / ゲームパッド右スティック
- 仮攻撃A: マウス左ボタン / ゲームパッドX

このビルドの目的
----------------
Gate 0承認後のPhase 1技術経路だけを検証するオフライン試作です。
素材、正式CombatForm、快斬・重断、変形、魔法、ボス、ステージ、MMO通信、
アカウント、課金、本番アート・音響はまだ実装していません。

同梱物
------
- MFO-Phase1.exe: Windows x86_64ポータブル実行ファイル
- RUN-MFO-PHASE1.txt: この説明
- GODOT-LICENSE.txt: Godot Engineのライセンス表示

注意
----
これは開発途中の技術試作であり、完成ゲームではありません。
実ゲームパッドでの操作感はGate 1の手動確認項目です。
