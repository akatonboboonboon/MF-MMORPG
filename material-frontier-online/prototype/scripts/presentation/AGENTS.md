# Presentation implementation instructions

このディレクトリでは、ルート `AGENTS.md` に加えて次を守る。

- 表示は `DomainEvent` と読み取り専用状態を消費し、シミュレーション状態を書き換えない。
- HP、`Integrity`、`Deformation`、部位破壊、命中、報酬、ギミック結果を表示側で決定しない。
- VFX、パーティクル、光、音、画面振動、ヒットストップの有無や長さを独自にゲームルール化しない。
- `docs/ASSET_CONTRACTS.md` で `Approved` または `Current` でないイベント応答を実装しない。
- 攻撃予告、プレイヤー輪郭、部位状態、ギミック状態、経路は最低画質でも消さない。
- 背景エフェクトと攻撃予告の色・形を混同させず、前景でプレイヤーを完全に隠さない。
- 装飾物へ当たり判定を付けない。ゲームプレイ上の意味がある例外は監督承認を必要とする。
- 美術スタイル、基準カメラ、仮音響、可読性の未決事項（OD-040～043）を推測実装しない。

最低検証: 通常・グレースケール・色覚補助・最低画質の可読性、およびパーティクル無効時の戦闘結果不変。
